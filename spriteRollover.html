Simple CSS Rollover

* REMINDER
There is no need to use JavaScript to do simple rollovers.
 

Creating a class in CSS

.spriteAnimation{
    width:40px; /* This is the width of the sprite */
    height:40px; /* This is the visible height of the sprite */
    max-height:200px; /* This is the height of the sprite's source image */
    background:url('../img/spriteAnimation.png'); /* This is the sprite's source image */
}

Add the spriteControl.js file

<script src="js/spriteControl.js"></script>

Initiating Sprites

kcSpriteInit(OBJECT_TO_ANIMATE, IS_BUTTON, ANIMATION_SPEED);

EXAMPLES:

kcSpriteInit($("#myDivAnimatedA"), true, 600);
 

kcSpriteInit($("#myDivAnimatedB"), true, 100);
 

kcSpriteInit($("#myDivAnimatedC"), false, 200); /* Not a button */
